# 字符串类型
字符串方法都不会修改本身。
## 字符方法
- charAt()
获取指定位置的字符
- charCodeAt()
获取指定位置的字符编制
    ```
    var a = '123'
    a.charCodeAt(1)
    // 50
    ```
## 操作方法
- concat()
拼接字符串，返回新字符串
- slice(start,end)
切分字符串，可以传入负数（会用数组长度加上该负数来确定响应位置）。
- substr(start,length)
切分字符串，第二个参数为往后截取的个数。
- substring(start,end)
切分字符串,传入的负数统一为0

## 位置方法
- indexOf()
从前往后查找，查到返回位置，未找到返回-1
- LastIndexOf()
从后往前查找，查到返回位置，未找到返回-1

## 格式化方法
- trim()
删除前置和后缀的所有空格。
## 大小写切换方法
- toUpperCase()
转大写
- toLowerCase()
转小写

## 匹配方法
- match()
根据传入的正则对象，返回第一个匹配正则表达式的的字符串。
- search()
根据传入的正则对象，返回第一个匹配正则表达式的的字符串的索引。找不到返回-1
