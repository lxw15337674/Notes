# 语法
typescript 作为一个带有类型检查器的 js 编译器，

## 特点
1. 强类型语言要求
    1. 变量，参数都要声明数据类型
    2. 增加接口、类型，用于对象的抽象
    3. 文件声明
2. 静态检查
    1. 错误提示
    2. 参数提示
